# Drawing
Rust project that generates a PNG image by drawing different geometrical shapes using the `raster` crate.

## Features
- Draw random points
- Draw random circles
- Draw rectangles
- Draw triangles
- Draw lines
- Trait-based design (`Drawable`, `Displayable`)

## Dependencies
- raster
- rand

## Run
```bash
cargo run
```

After execution, an `image.png` file will be generated.
